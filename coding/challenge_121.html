Task 2: The Travelling Salesman

So much similar to Task 2 in Week 118. However, after I read the optimized solution from James Smith's, after I read resources on A* search, after I read Abigail's solution, I am still puzzled and do not have a clear idea to tackle this task. So I have just been lazy and use permutations again: 

 
my $v = chr(ord("A")+$N-2);
my $iter = permutations(['A'..$v]);
while (my $c = $iter->next) {
    my $d = calc_dist($c);
    if (defined($d) && $d <= $min) {  # [*]
        if ($d < $min) {
            @arr_min = ();
            $min = $d;
        }
        push @arr_min, $c;
    }
}
